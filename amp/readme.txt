README.TXT for AmpJuke.

...quick&dirty INSTALLATION:

1. Download AmpJuke, extract the files in the "AmpJuke-x.x.x.tar.gz"-archive,
and - if you need to do so - use Filezilla or a simliar FTP-related program to 
transfer the contents to your webserver.

2. Open a browser and point it to: http://www.your-server.com/ampjuke-directory/

3. Follow the onscreen instructions by entering the fields in the form.



More resources: Please see the homepage http://www.ampjuke.org
...especially the FAQ/documentation as well as the installation example.
